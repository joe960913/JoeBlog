antd-table组件添加分页属性

Nov 09, 2018 10:44· 70 words · 1 minutes read

table 中添加分页属性

定义分页属性

获取model中后台的数据data

total:data.total

pageSize:data.pageSize

current（表示当前页数）:data.pageNum

onChange（页码改变触发的func）:(current)=>(this.changePage(current))

定义changePage:

changePage=(current)=>{
  const { dispatch } = this.props
    dispatch({
    type:'query'
    payload:{pageNum:current}
  })
}
