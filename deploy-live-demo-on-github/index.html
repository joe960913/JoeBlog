关于如何打包React项目将其展示在GitHubPage上

Author: Joe Date: December 16, 2019  11:19:30 Category: Work

  1. 将你项目和github仓库连接起来

  2. 确保你的项目可以成功跑起来

  3. 修改项目中的package.json文件

    {
  "name": "my-weather",
  "version": "0.1.0",
  "private": true,
  "homepage": "https://joe960913.github.io/My-Weather-React",
  "dependencies": {
  ...
  }
}

    添加homepage属性，value为https://你的github用户名.github.io/你的项目仓库名

    "scripts": {
    "start": "node scripts/start.js",
    "build": "node scripts/build.js",
    "test": "node scripts/test.js",
    "predeploy": "npm run build",
    "deploy": "gh-pages -d build"
  },

    在脚本内容中添加predeploy和deply属性

  4. 安装gh-pages

    npm install gh-pages --save-dev

  5. 部署项目

    npm run deploy

  6. 进入仓库，会多一个gh-pages分支

    进入仓库settings，检查项目是否已经部署到https://joe960913.github.io/xxxx
